## Flexbox

### Часто используемые кейсы:

**Контейнер с отступами по краям**



**Адаптивная сетка**

Это такая сетка, которая подстраивает количество столбцов и строк в зависимости от разрешения экрана. То есть на экране смартфона сайт будет выглядеть иначе, чем на экране ноутбука или планшета, чтобы пользователю было комфортно взаимодействовать с интерфейсом. Такие сетки используются, например, для карточек товаров и интернет-магазинах.

<TabItem value="flexbox" label="CSS Flexbox">
<div style={{height: '380px', overflow: 'auto'}}>

```css
.container {
  max-width: 700px;
  margin: 0 auto;
}

.grid {
  display: flex;
  flex-wrap: wrap;
  flex-direction: row;
}

.item-wrap {
  box-sizing: border-box;
  padding: 10px;
  height: 150px;
  flex-basis: 25%;
}

.item {
  background: #aaa;
  height: 100%;
  width: 100%;
}

@media (max-width: 600px) {
  .item-wrap {
    flex-basis: 33.33%;
  }
}

@media (max-width: 500px) {
  .item-wrap {
    flex-basis: 50%;
  }
}

@media (max-width: 400px) {
  .item-wrap {
    flex-basis: 100%;
  }
}
```

</div>
</TabItem>

<TabItem value="grid" label="CSS Grid">
<div style={{height: '380px', overflow: 'auto'}}>

```css
.container {
  max-width: 700px;
  margin: 0 auto;
}

.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 20px;
}

.item-wrap {
  box-sizing: border-box;
  height: 150px;
  background: #aaa;
}

@media (max-width: 600px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

@media (max-width: 500px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 400px) {
  .grid {
    grid-template-columns: repeat(1, 1fr);
  }
}
```
